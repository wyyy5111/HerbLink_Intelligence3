## 🔧 如何使用

### 启动开发服务器
```bash
cd frontend
npm run dev
```
访问: http://localhost:5176/

### 构建生产版本
```bash
cd frontend
npm run build
```

### 修改品牌颜色
编辑 `frontend/src/assets/styles/brand.scss` 中的CSS变量：
```scss
:root {
  --color-primary-600: #22c55e;  // 主色
  --color-secondary-500: #0ea5e9; // 辅助色
  // ...
}
```

### 添加新动画
在 `frontend/src/utils/animations.ts` 中添加：
```typescript
export const myAnimation = (selector: string) => {
  gsap.fromTo(selector,
    { opacity: 0 },
    { opacity: 1, duration: 0.6 }
  )
}
```

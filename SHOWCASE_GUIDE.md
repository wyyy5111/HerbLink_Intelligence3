## 🎮 交互操作

### 1. 切换平台
- 点击顶部导航的 **"农户端"** 或 **"客户端"** 按钮
- 或使用快捷键：
  - `Alt + 1` = 切换到农户端
  - `Alt + 2` = 切换到客户端

### 2. 卡片交互
- **鼠标悬停** → 卡片上浮 + 3D倾斜效果
- **点击卡片** → 波纹扩散动画

### 3. 滚动查看
- 向下滚动查看所有功能模块
- 卡片会依次滑入，错落有致

### 4. 全屏展示
- 点击右上角的全屏按钮

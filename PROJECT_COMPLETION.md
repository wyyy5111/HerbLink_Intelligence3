## 🚀 快速开始

### 方式1：直接打开
```bash
# 农户端
open mobile-farmer.html

# 客户端
open mobile-client.html
```

### 方式2：本地服务器
```bash
# 启动服务器
python -m http.server 8000

# 访问
# http://localhost:8000/mobile-farmer.html
# http://localhost:8000/mobile-client.html
```

### 方式3：移动设备预览
```bash
# 1. 确保设备和电脑在同一Wi-Fi
# 2. 启动服务器
python -m http.server 8000

# 3. 在移动设备访问
# http://[电脑IP]:8000/mobile-farmer.html
```
